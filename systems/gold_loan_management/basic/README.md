# Gold Loan System - Basic Edition

Auto-generated by ZORO9X SaaS Platform

## Features

### User Roles
- **Cashier**: Create loans, search/view loans, renew, redeem, print tickets, manage customers
- **Admin**: All cashier permissions + manage market rates, interest rates, duration settings, user management, company settings

### Core Features
- **New Loan Ticket**: Customer details, gold article entry, auto-calculate market/assessed values
- **Loan Management**: Search, view details, full financial breakdown  
- **Renew Loan**: Extend loan duration with interest payment
- **Redeem Loan**: Full redemption with interest + overdue calculation
- **Customer Management**: Add/edit customers, view customer loan history
- **Reports**: Overdue loans, active portfolio, redeemed loans, summary
- **Print**: A4 and Receipt (1/3 A4) format printing via browser

### Admin Settings
- **Market Rates**: Set Rs./8 grams (1 poun) for each carat (16K-24K)
- **Duration Rates**: Configure assessed value %, interest rate, and overdue rate per duration
- **User Management**: Create/edit cashier and admin users
- **Company Settings**: Company name, phone, address, ticket prefix

### Business Logic
- Market Value = Gold Weight × (Market Rate per 8 grams / 8) (by carat)
- Assessed Value = Market Value × Assessed Percentage (by duration)
- Loan Amount = Assessed Value
- Interest = Loan Amount × Rate × Duration
- Overdue Interest = daily rate × overdue days
- Currency: Rs. (Rupees)

## Installation

1. Build the installer: run `BUILD.bat`
2. Share only `dist/gold_loan_installer.exe` with end users
3. User runs the installer and enters install path, API key, and setup info
